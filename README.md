# LoL Esport Manager

Simule une carrière de manager esport dans l'univers de League of Legends.
Gère ton équipe, prépare tes drafts, négocie des contrats et mène ton club vers les Worlds.

## Téléchargement

👉 **[Voir la dernière version](https://github.com/ZeqoAe/LOLM-RELEASE/releases/latest)**

---

## Installation

### 🪟 Windows
1. Télécharge `LoL.Esport.Manager_x.x.x_x64-setup.exe`
2. Double-clique sur le fichier
3. Si Windows affiche un avertissement "application inconnue", clique sur **"Informations complémentaires" → "Exécuter quand même"**
4. Suis l'installeur

### 🍎 macOS
1. Télécharge `LoL.Esport.Manager_x.x.x_aarch64.dmg`
2. Double-clique sur le `.dmg` pour l'ouvrir
3. Glisse l'application dans ton dossier **Applications**
4. Au premier lancement, fais **clic droit → Ouvrir** (nécessaire une seule fois car l'app n'est pas signée)

### 🐧 Linux
**AppImage (recommandé, aucune installation) :**
1. Télécharge `LoL.Esport.Manager_x.x.x_amd64.AppImage`
2. Rends le fichier exécutable : `chmod +x LoL.Esport.Manager_*.AppImage`
3. Lance-le : `./LoL.Esport.Manager_*.AppImage`

**Paquet .deb (Ubuntu / Debian) :**
1. Télécharge `LoL.Esport.Manager_x.x.x_amd64.deb`
2. Installe : `sudo dpkg -i LoL.Esport.Manager_*.deb`

### 🤖 Android
1. Télécharge `app-debug.apk`
2. Dans les paramètres Android : **Sécurité → Sources inconnues → Autoriser**
3. Ouvre le fichier `.apk` et installe
